This is the source code for the Apache Logging Services website hosted at following locations:

* https://logging.apache.org[] (production)
* https://logging.staged.apache.org[] (staging)

The project uses https://logging.apache.org/logging-parent[logging-parent] as the parent Maven project to build the website.

[#requirements]
== Requirements

* JDK 17+
* A modern Linux, OSX, or Windows host

[#build]
== Building

You can build the website as follows:

[source,bash]
----
./mvnw site
----

You can view the generated website with a browser by pointing it to the `target/site` directory.
