Remarks

This method first checks if this logger is WARN enabled by reading the value property. This check happens always and does not depend on the implementation. If this logger is WARN enabled, then it converts the message object (passed as parameter) to a string by invoking the appropriate IObjectRenderer. It then proceeds to call all the registered appenders in this logger and also higher in the hierarchy depending on the value of the additivity flag.

WARNING Note that passing an Exception to this method will print the name of the Exception but no stack trace. To print a stack trace use the WarnExt form instead.
