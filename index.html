Apache Logging Services

The Apache Logging Services project creates and maintains open-source software related to the logging of application behavior and released at no charge to the public.

Subprojects

Log4j®

A versatile, industrial-grade Java logging framework composed of an API, its implementation, and components to assist the deployment for various use cases.

Log4j® Kotlin

A Kotlin-friendly interface to log against the Log4j® API.

Log4j® Scala

A Scala-friendly interface to log against the Log4j® API.

Log4cxx

A logging framework for C++ patterned after Log4j®.

Log4Net

A port of the Log4j® to the Microsoft .NET runtime.

Click to see list of dormant subprojects
Chainsaw

A GUI-based log viewer intended as a companion to Log4j®.

Flume

Distributed, reliable, and available service for efficiently collecting, aggregating, and moving large amounts of log data.

Log4j® 1

The very first Log4j® major release. Reached End-Of-Life in August 2015.

Log4j® Audit

Audit logging framework built upon Apache Log4j®.

Log4j® Extras

Extras for Log4j® 1; companions, receivers, and more.

Log4php

A versatile logging framework for PHP. Originally a port of Log4j® to PHP, it has grown to include various PHP specific features.
